## 🚀 Local Terminal Setup

If you want to pull down the telemetry matrix and adjust the core code parameters locally:

1. **Clone the repository to pull down all matrix assets and system files:** 
```bash
 git clone https://github.com/Vikas-Verma1/Cyborg-Themed-Landing-Page.git
```
 
2. **Verify Directory Layout**
   Ensure your directory structure mirrors the required relative paths so internal components load seamlessly:
   ```text
   ├── index.html   # Main system markup and styled layers
   └── image.png    # Cybernetic UI graphic asset (Cyborg Image)
   ```

3. **Run the Interface**
   Initialize the interactive client on your local engine using either tool chain:
   * **Method A (VS Code Live Server):** Open the project folder in VS Code, right-click `index.html`, and select **Open with Live Server**.
   * **Method B (Python Environment):** Spin up an instantaneous localized server directly via your system terminal:
     ```bash
     python -m http.server 8000
     ```
     Once initialized, direct your web browser pathing to: `http://localhost:8000`
## Most Recommended Method :

 1. Open **VS Code** and click on the **Extensions** icon on the left sidebar (or press `Ctrl+Shift+X` / `Cmd+Shift+X`).
